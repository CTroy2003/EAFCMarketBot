# EAFCMarketBot
THIS IS A WORK IN PROGRESS

The purpose of this program is to quickly buy players below market value on the EAFC 25 marketplace. We use selenium to access the EAFC 25 web app and navigate to the transfer market. Im using Selenium combined with the chromedriver and the chrome testing browser. As im running this locally on a mac and chrome there might need to be some slight adjustments made based on your preffered OS and browser.

TO RUN LOCALLY FOLLOW THESE STEPS:

1. Adjust the necessary file paths and email/password information.
2. When prompted with an email code, enter this into the console.
